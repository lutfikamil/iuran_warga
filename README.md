# Iuran Warga

Aplikasi ini untuk management keuangan dan data warga perumahan, bisa juga untuk RT dan RW + Laporan keuangan Mushollah di lingkungan warga.

## Beberapa Fitur

-Management iuran warga.

-Pendataan warga dan history warga.

-Laporan Rekapitulasi dan detail ke warga.

-Iuran yang di pungut Petugas iuran tetap tercatat tapi butuh validasi Bendahara untuk menyatakan uang sudah di terima Bendahara.

-Tagihan otomatis terkirim ke WhatsApp ketika generate.

-Laporan keuangan Musollah sebagai Informasi keuangan ke warga.
